# Linux System Monitoring Tool

A Bash-based Linux monitoring utility that automates server health checks and provides a quick overview of system performance and resource utilization.

## Features

- CPU Monitoring
- Memory Usage Monitoring
- Disk Usage Monitoring
- System Uptime Tracking
- Running Process Analysis

## Tech Stack

- Linux
- Bash Scripting
- Git
- GitHub

## Use Case

This tool helps system administrators and DevOps engineers quickly check server health and identify resource-related issues without running multiple commands manually.
